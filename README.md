# -Rock-Paper-Scissors
🎮 Rock Paper Scissors Game built with HTML, CSS, and JavaScript. Play against the computer with random moves, real-time score tracking, and instant win/lose/draw results. Features an interactive UI, dynamic feedback, and responsive design while showcasing JavaScript DOM manipulation, event handling, and game logic.
Features
Play Rock, Paper, Scissors against the computer
Random computer move generation
Real-time score updates
Win, Lose, and Draw detection
Interactive and user-friendly interface
Dynamic result messages with visual feedback
Responsive design for different screen sizes
🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)
